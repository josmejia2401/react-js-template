# REACT JS TEMPLATE/PLANTILLA
Plantilla básica.
# Nuevas caracteristicas
  - ReactJs
  - Redux v.4.0.5
  - Chartjs-2 v.2.9.0
  - React v.16.13.1
# Dependencias
Para instalar las dependencias:
```
npm install
```
# Ejecución
Para ejecutar el proyecto:
```
npm start
```
# Compilación para ambiente PRODUCTIVO:
Para ejecutar el proyecto:
```
npm run build_public_url_jgmm
```
